Honest notes on this build

  • Demo vs Live data: the demo dashboard runs on synthetic sample data (always available); the live dashboard runs on real Apify data for the configured brands (refreshed weekly via the GitHub pipeline).
  • Time window: scraping defaults to the last 7 days of mentions. Reddit returns consistent recent results; TikTok may surface older top videos; Instagram timestamps need normalization in this version.
  • Channels covered: Reddit, TikTok, Instagram. News, blogs, Facebook, YouTube require additional setup (v2).
  • What it does NOT do (yet): no real-time streaming, no paid-API access for Instagram/Facebook, no multi-language sentiment (English-only model), no automated alerting to Slack/email.

Built as a personal portfolio piece to showcase how AI augments — not replaces — the work of a senior marketing consultant.
