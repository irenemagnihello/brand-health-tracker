Case Study #1

AI-Assisted Brand Perception & Signal Monitoring

Agile social listening and early-warning intelligence for consumer & beauty brands.

Why social listening matters

Tracking what people say about a brand on social media is essential — it surfaces perception shifts, complaint patterns, and product feedback weeks before any survey would catch them. But getting reliable cross-platform data at scale is hard. Most platforms restrict scraping, sentiment APIs are costly, and real-time monitoring tools charge premium prices.

This is a replicated version of the process I used to run manually at TLGG (Mercedes) and TD Reply (BMW, L'Oréal). I set it up myself using Python, Apify, and Claude. Limitations are visible in the data — for a more accurate analysis, significantly more effort is required.

The AI-Augmented Setup

  • Multi-source ingestion: scrapes mentions across TikTok, Reddit, Instagram.
  • LLM processing: Claude evaluates sentiment, topics, crisis signals per mention.
  • Executive output: Looker Studio dashboard + 2-page weekly PDF briefing.

Scope & limitations

  • Channels: Reddit, TikTok, Instagram. News / blogs / Facebook / YouTube require additional setup.
  • Time window: defaults to last 7 days; TikTok may surface older top videos.
  • Language: English-only model. No real-time streaming, no paid API access for Instagram/Facebook.
